Companies House personal code

What a Companies House personal code is

This is an 11 character code that is given to a person once they’ve verified their identity.

The code is personal to you, not your company or the company you work for.

Why you need a personal code/what to do with it

You will need to use your Companies House personal code to connect your verified identity to our records. This is so that we know the correct person is linked to the name we have on file.

Keeping your code secure

You may need to share this code with people who file on your behalf, or for your company. Only share the code with people you trust.

Keep this information secure, as you would with other unique codes such as your Unique Taxpayer Reference (UTR) for HMRC.

Explaining the difference between personal code and authentication code

Your personal code is not the same as the company authentication code. If you have an authentication code, you should continue to keep this safe.

Where to find the Companies House personal code

This is an 11 character code that is given to a person once they have verified their identity.

You can find your code in the ‘Manage account’ section of your Companies House account. You’ll need to sign in using the email address you provided when you verified.
