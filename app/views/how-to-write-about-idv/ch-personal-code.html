Companies House personal code

What a Companies House personal code is

This is an 11 character code that is given to a person once they’ve verified their identity. The code is personal to you, not your company or a company you work for.

Why you need a personal code and what to do with it

You'll need your personal code for various reasons. For example:

  • as part of some company filings, such as the confirmation statement (opens in new tab)
  • if you are appointed as a director
  • if you become a person with significant control (PSC) (opens in new tab)

If you are a director or a PSC, you'll need to use your Companies House personal code to connect your verified identity to our records. This is a requirement, and will ensure that we know the correct person is linked to the name we have on file.

Keeping your code secure

Only share the code with people you trust

You may need to share this code with people you trust to file on your behalf, or for your company. Keep this information secure, as you would with other unique codes such as your Unique Taxpayer Reference (UTR) for HMRC.

Identity verification makes it much more difficult for criminals to impersonate someone, but not impossible.

Explaining the difference between personal code and authentication code

Your personal code is not the same as the company authentication code. If you have an authentication code, you should continue to keep this safe as you'll need it to file online. You may need your personal code for some online filings as well.
