Description

IDAM stands for Identity Access Management. The project is responsible for the implementation of GOV.UK One Login, which is being rolled out across all government departments.

In the future, users will be able to use GOV.UK One Login using the same email address and password to access all government services. GOV.UK One Login will eventually replace the current way to sign in to a Companies House account and WebFiling.
